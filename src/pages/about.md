Electric Lime is a modern, accessible 11ty starter kit designed to help you build beautiful, fast websites with minimal effort.

## Our Mission

We believe that building websites should be simple, fast, and enjoyable. That's why we created Electric Lime - a starter kit that combines the best tools in the modern web development ecosystem.

## Technology Stack

### Eleventy (11ty)
A simpler static site generator that transforms templates into HTML. No client-side JavaScript required for core functionality.

### Tailwind CSS v4
The latest version of the utility-first CSS framework with improved performance and the new @theme system for seamless customization.

### Alpine.js
A lightweight JavaScript framework for adding interactivity to your pages without the overhead of larger frameworks.

### Nunjucks
A powerful templating engine with features like template inheritance, macros, and filters.

## Features

- **Accessible by default** - WCAG AA compliant with high contrast mode
- **SEO optimized** - Meta tags, Open Graph, sitemap, and RSS feed
- **Performance focused** - Static HTML, optimized images, minimal JavaScript
- **Developer friendly** - Hot reload, modular architecture, clear documentation
